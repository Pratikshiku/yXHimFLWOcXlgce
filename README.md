## Stray Animal Rescue Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/48b8a7b29e014ef786a94208cb8ba72e/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Animal Adoption and Donation
5. Forum
6. Announcement Information
7. Products
8. Pet Finding Requests
9. Personal Center
10. Delivery Address
11. Animal Favorites
12. Animal Adoption Review
13. Product Orders
11. Shopping Cart


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Animal Adoption, Donation Management, Animal Message Management, Animal Favorites Management
5. Animal Adoption Review Management, Announcement Type Management, Product Type Management
6. Basic Data Management, Animal Type Management, Announcement Type Management, Product Type Management
7. Forum Management
8. Announcement Information Management
9. Product Management, Product Review Management, Product Order Management
10. Pet Finding Request Management
11. User Management
12. Carousel Management
